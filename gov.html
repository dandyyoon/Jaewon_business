02 · GOVERNMENT GRANT MANAGEMENT

참여율만 입력하면,
정산이 자동으로 맞춰집니다

정부 R&D 과제의 예산 편성부터 인건비·현물 자동계상, 집행·증빙, 정산·보고까지 전 과정을 하나의 웹앱에서. 정산 지적 단골 항목을 시스템이 자동으로 경고합니다.

● 실무 현장에서 운영·검증된 시스템
기술 · React SPA · Supabase(PostgreSQL + Auth) · SheetJS 정산 엑셀 · Google Drive 증빙 연동
인건비·현물
= 자동 계상수작업 계산 제거

정부과제 정산에서 가장 손이 많이 가고 실수가 잦은 인건비·장비 현물·임차 현물. 이 시스템은 직접 집행만 입력하면 참여율·취득가·임차료를 기준으로 나머지를 자동 계산합니다. 매월 반복되는 인건비 계상은 정해진 날짜에 자동 반영됩니다.

Workflow

과제 라이프사이클 전 과정

대시보드 → 등록·설정 → 수행·집행 → 정산·보고, 4단계로 과제를 관리합니다.

📊
STEP 0

대시보드

  • 총 과제 수·총 사업비·총 집행액 KPI 총괄
  • 과제 목록 카드/표 보기, 예산 대비 집행률
  • 여러 과제의 예산을 한 화면에서 조망
🗂️
STEP 1

등록 · 설정

  • 과제 등록(주관기관·사업명·협약기간·상태·증빙폴더)
  • 협약기간 기준 연차 자동 생성(1차·2차년도…)
  • 사업비 계산기 — 정부지원금·자부담 비율 역산
  • 세부 비목 등록, 총합 = 총사업비 자동 검증
⚙️
STEP 2

수행 · 집행

  • 인력 배정 — 직원·과제·월별 참여율 매트릭스
  • 인건비 자동계상(참여율 × 월급여, 천원 절사)
  • 현물 관리 — 장비 감가상각·임차 배분 자동 계산
  • 집행 입력(공급가액·부가세·거래처·증빙 URL)
📁
STEP 3

정산 · 보고

  • 연차별·비목별·재원별 집행 현황 집계
  • 비목 표준 폴더 자동 생성으로 증빙 정리
  • 정산 엑셀 원클릭 내보내기
  • 감사 로그(누가·언제·무엇을) 자동 기록
Compliance built-in

정산 지적을 시스템이 미리 막습니다

정부과제 규정을 규칙으로 내장해, 실수·환수 리스크를 사전에 경고합니다.

⚠️ 자동 경고 규칙

참여율 월 합산 100% 초과 동일 장비 중복 배정 비목간 변경 20% 한도 초과 임차 배분율 초과 비목 총합 ≠ 총사업비

여러 과제를 동시에 수행하며 참여연구원이 겹치기 쉬운 의료기기·바이오 R&D 기업에 특히 효과적입니다. 주요 정부 R&D 사업유형별 권장 비목 비율도 반영할 수 있어, 신규 과제 예산 편성 시 가이드로 활용할 수 있습니다.

Data model

관계형 DB 구조

과제·연차·비목·인력·집행·장비·임차가 서로 연결된 10개 테이블로 설계되었습니다.

테이블대표 항목
과제 / 연차사업명·주관기관·협약기간·상태·총/정부/현금/현물 예산 · 연차별 예산·이월금
비목비목명·재원별 예산·유형(자동인건비/자동장비/자동임차/직접)
인력 배정과제·직원·참여율·월별율(1~12월)·재원구분 — 인건비 자동계산 핵심
집행 내역일자·비목·공급가액·부가세·거래처·지급방법·증빙 URL·입력자·입력시각
장비 / 임차취득가·내구연한·사용개월·현물액(자동)·중복플래그 · 월임차료·배분율·배분기준
감사 로그시각·사용자·행위(등록/수정/삭제)·대상 — 최근 이력 추적
Why it sells

도입 효과

정산 실수·환수 리스크 제거

참여율 초과·장비 중복·비목 변경 한도 등 정산 지적 항목을 자동 경고. 감사·실사 대응력이 올라갑니다.

회계 담당 수작업 대폭 절감

참여율만 입력하면 인건비·현물이 자동 산출·집행 반영. 매월 반복 계상 작업이 사라집니다.

다과제 통합 가시성

여러 과제의 예산·집행·잔액·집행률을 하나의 대시보드로. 과제별 엑셀을 하나의 DB로 통합.

증빙·정산 자동화

비목별 표준 폴더 자동 생성으로 증빙을 규정대로 정리하고, 정산 엑셀을 즉시 출력합니다.
